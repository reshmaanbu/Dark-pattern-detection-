About Dark Pattern Detector

What is this?

Dark Pattern Detector is a free, open-source tool that analyzes websites for deceptive UI patterns — techniques that companies use to manipulate your decisions. We believe in a more honest, transparent web.

How does detection work?

We use a rule-based NLP approach: the page text is scraped, cleaned, split into sentences, and matched against a curated keyword database for four pattern categories. Each match contributes to a deduction from the base Honesty Score of 100.

Fake Urgency match−10 pts
Scarcity match−8 pts
Social Pressure match−6 pts
Confirmshaming match−7 pts

Tech Stack

Python / Flask BeautifulSoup SQLite Chart.js Vanilla JS pytesseract (optional)

Limitations

This tool uses text-based heuristics and may produce false positives or miss visual-only dark patterns. It is a starting point for awareness, not a legal or definitive audit. Always use your own judgment.
